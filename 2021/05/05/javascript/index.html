JavaScript 学习笔记

JavaScript 学习笔记

  • 特点

    • 解释执行，(一句一句编译)
    • 灵活，动态特性，可以随意给对象增加属性和方法
    • 头等函数，函数在JS中是一等公民
    • 执行环境，宿主环境（浏览器、Node）

  • 作用

    • JavaScript主要用于读写HTML元素、在网页中嵌入动态文本、动态修改CSS样式表；
      对浏览器事件做出响应、表单数据验证、检测访客的浏览器信息等等

  • 构成

    • 核心 ( ECMAScript )
    • 文档对象模型 ( DOM )
    • 浏览器对象模型 ( BOM )

一、 ECMAScript

  • 严格模式 “use strict”

  • 推荐末尾加上分号；

变量

  • 变量是一个‘容器’
  • 变量存储在内存中，简单数据存在栈内容中，复杂数据存在堆内存中

基本类型

Array
  • push()
    • 在数组的最后添加一个元素

事件流的三个阶段

  • 捕获：从外向内
  • 执行目标阶段
  • 冒泡：从内向外

二、DOM

  • 常见的输入输出语句
    • alert()：浏览器弹出提示框，输出
    • document.write()：向页面输出内容
    • prompt()：弹出一个对话框，让用户输入

JS模式

  • 单例模式
    • 需要创建多个对象的场景下，通过一个变量接收，始终保持只有一个对象
    • 节省内存空间
